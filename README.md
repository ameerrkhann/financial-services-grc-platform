# Financial Services Cyber Risk Intelligence Platform

**Author:** Ameer Mohammad Khan — 3rd Year CS, University of Toronto  
**Status:** 🔨 In Progress (30-Day Build)

---

## What This Project Is

A Python-based GRC portfolio project simulating a real cybersecurity risk 
program for a fictional Canadian financial institution.

## Three Modules

| Module | Description | Frameworks |
|--------|-------------|------------|
| NIST CSF 2.0 Assessment | Maturity scoring engine with gap analysis | NIST CSF 2.0, ISO 27001, SOC 2 |
| Quantitative Risk Engine | Dollar-value risk modeling across 5 threat scenarios | FAIR, Netflix riskquant |
| Vendor Risk Assessor | Questionnaire generator + risk tiering | OSFI B-10, B-13 |

## Tech Stack
Python · SQL/SQLite · Power BI · Streamlit · pandas · matplotlib
